Sírius ERP – Cadastro de Clientes

Controle completo de seus clientes.

Cadastre seus clientes

  • Alguns recursos do módulo:
  • Endereço de cobrança
  • Endereço de entrega
  • Cadastro de diversos locais de entrega para o mesmo cliente
  • Limite de crédito
  • Bloqueio do cliente
  • Vínculo com vendedor
  • Vínculo com transportadora
  • Vínculo com tabela de preço
  • Mini CRM para registro de contatos com o cliente
  • Conta corrente financeira do cliente
  • Verificação de CNPJ, CPF e Inscrição Estadual
  • Carregamento automático do endereço a partir do CEP
  • Carregamento automático dos dados cadastrais a partir do CNPJ
  • Impressão:
  • Imprima etiquetas utilizando os dados cadastrais de seus clientes
  • Faça diversos relatórios utilizando os dados de clientes cadastrados. Alguns relatórios podem ser gerados a partir do filtro escolhido
  • Filtros:
  • CNPJ
  • Endereço
  • Razão social
  • São aproximadamente 30 opções de filtragens além da filtragem composta.
  • Sem contrato de fidelidade para planos mensais.
